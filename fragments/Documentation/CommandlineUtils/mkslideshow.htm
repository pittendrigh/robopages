mkslideshow creates a slideshow but it does NOT create thumbnail links If you want thumbnail navigation links see mkthumbs.htm

To run mkslideshow ssh into the server (or run it on localhost for development). Cd to the directory where you want the slideshow.
type:
mkslideshow

Note that mkslideshow, as written, requires sudo permission, so it runs on virtual dedicated servers where the apache installation is picky about file permissions.
