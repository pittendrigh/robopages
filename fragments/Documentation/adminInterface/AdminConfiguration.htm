Setting up the AdminInterface

Edit conf/globals.php so $sys_robosauce = "whatEverYouWant";


Also hand edit commandLineUtils/phppw
so $robosauce="whatEverYouWant";
Those two have to match.

now use commandLineUtils/phppw:

phppw apassword >> plugins/RobopagePasswds.php <enter>

now edit plugins/RobopagePasswds.php so the appended last line has the form:

username:hash:nimda
, where "hash" is the output from commandLineUtils/phppw
Now point your browser at https://yourdomain.net/?layout=nerd

You will be prompted for a password.
The login should work (on localhost only) without resetting the password,
using login: gitgit with password: gotgot

gitgit and gotgot is fine on your localhost but you must reset the password for real use on any server.
