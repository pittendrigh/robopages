Installing into a unix file system public_html directory

In the main index.php include and instantiate dirCrawler.class.php instead of robopages.php

Then edit conf/dirLStart.conf.ini

include_once("dirCrawler.class.php");
//include_once("robopages.php");

$page = new dirCrawler(); >
//$page = new robopages()


echo $page->startHTML($title);
echo $page->crawl();
echo dirCrawler::endHTML();
