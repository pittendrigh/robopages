Copy and image to this direcotry and/or edit a somename.htm file here and see what happens to hte navigation ast screen left.

From a terminal window on linux do a mkdir -p roboresources/pics from this directoy. Copy an appropriately sized image to roboresoruces/pics
Now edit somename.htm so it includes:

<img src="_ROBOPATH_roboresources/pics/somename.jpg" alt="whatever" />'

_ROBOPATH_ in the img element above gets expanded to the url of the current directory, so when you move the whole directory the images (in any *.htm file) go with the move.
